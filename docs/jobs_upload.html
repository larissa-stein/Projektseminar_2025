Jobs_upload module

src.jobs_upload.load_job_titles()[source]

Load saved job titles from a local JSON file.

This function reads job titles that were previously stored in a local JSON file. These job titles are typically defined by users in the admin dashboard and used for scraping job advertisements. The loaded list is made available to the dashboard upon application start.

Returns:

A list of job titles. Returns an empty list if the file is missing or unreadable.

Return type:

list

src.jobs_upload.save_job_titles(job_title_list)[source]

Save the provided list of job titles to a local JSON file.

This function overwrites the existing local job titles file with the given list. It is used after the user adds or removes job titles in the admin dashboard. The saved list is also used as the basis for uploading to the MongoDB database.

Parameters:

job_title_list (list) – A list of job titles to be saved.

src.jobs_upload.delete_job_title(job_title)[source]

Delete a specific job title from the list.

This function removes a single job title from the saved list if it exists. The updated list is then saved to the JSON file.

Parameters:

job_title (str) – The job title to remove.

src.jobs_upload.delete_all_job_titles()[source]

Delete all saved job titles.

This function clears the entire list of saved job titles by overwriting the JSON file with an empty list.
